Question 11

What is an event loop and call stack

The call stack is a mechanism that helps the JavaScript interpreter to keep track of the functions that a script calls. Every time a script or function calls a function, it's added to the top of the call stack. Every time the function exits, the interpreter removes it from the call stack. The call stack is a LIFO queue (Last In, First Out).
The event loop continuously checks the call stack to see if there's any function that needs to run.
While doing so, it adds any function call it finds to the call stack and executes each one in order.
